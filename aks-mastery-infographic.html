From Zero to AKS Hero
Complete Learning Path: Docker → Kubernetes → AKS → Production Mastery
1Docker Fundamentals
Goal: Package applications + dependencies into portable images
Core Concepts
• Virtualization vs Containerization (VMs = OS+App | Containers = App+Libs)
• Docker Engine: Daemon + Client + Registry
• Images (read-only) vs Containers (running instance)

Key Commands & Files
• Dockerfile: FROM, RUN, COPY, CMD, ENTRYPOINT
• docker build • docker tag • docker push
• docker run/stop/exec/logs
• Volumes & Bridge networks

Checkpoint Lab: Build & run a Python/Node.js web app locally on port 8080
↓
2Kubernetes Fundamentals
Goal: Orchestrate containers at scale with self-healing & networking
Architecture
• Control Plane: API Server, Scheduler, Controller Manager, etcd
• Worker Nodes: Kubelet, Kube-proxy, containerd

Core Objects (YAML)
• Pod • ReplicaSet • Deployment (rolling updates)
• Service: ClusterIP, NodePort, LoadBalancer
• Namespace • ConfigMap • Secret

Checkpoint Lab: Minikube/Docker Desktop → Deploy your Phase 1 image with Deployment + Service
↓
3Azure Kubernetes Service (AKS)
Goal: Run production clusters with Azure-managed control plane
Azure Architecture
• Managed Control Plane (free) – you only pay for worker nodes
• Node Pools (System vs User pools)

Key Integrations
• Azure Container Registry (ACR)
• Entra ID (Azure AD) authentication
• Networking: Kubenet vs Azure CNI (pod gets VNet IP)

Checkpoint Lab:
1. Create ACR + push image
2. Create AKS cluster (az cli)
3. Grant AKS → ACR pull permission
4. Deploy your app to AKS
↓
4Advanced Production Operations
High Availability
• Availability Zones (span node pools across 3 zones)
• Pod Disruption Budgets (PDB)
• Cluster Autoscaler + Horizontal Pod Autoscaler (HPA)

Disaster Recovery
• Multi-region clusters (East US + West US)
• Azure Front Door / Traffic Manager failover
• Geo-replicated DBs (Cosmos DB, Azure SQL)

Backup & Restore
• Velero (open-source) + Azure Backup for AKS (managed Velero)

Monitoring & Observability
• Container Insights (Azure Monitor)
• Managed Prometheus + Managed Grafana
• Liveness & Readiness Probes
↓
5Expert & Security Topics
Security
• RBAC + Azure AD group mapping
• Azure Policy for Kubernetes (block public images)
• Secrets Store CSI Driver → Azure Key Vault

Ingress & Routing
• NGINX Ingress / Application Gateway Ingress Controller (AGIC)

GitOps
• Flux or ArgoCD – declarative Git-driven deployments
